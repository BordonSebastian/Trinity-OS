# Seguridad de la información

Este repositorio debe permanecer privado.

No almacenar:

- Contraseñas.
- Tokens.
- Claves API.
- Certificados `.p12`.
- Claves privadas.
- Datos personales sensibles.
- Copias completas de bases de datos.
- Información financiera confidencial de clientes.
- Credenciales de hosting o correo.

Usar gestores de contraseñas y variables de entorno para credenciales.
